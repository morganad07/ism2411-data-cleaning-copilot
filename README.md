This respository consists of a Python project that cleans a messy dataset. The final goal of this project is to show basic cleaning methods in code. It also shows the use of assistance of Copilot when creating codes.

The project includes the following files and folders: data, raw, sales_data_raw.csv, processed, sales_data_clean.csv,src, data_cleaning.py.

Pandas are used in the code and the code is used to clean a messy dataset. The project also includes a reflection on how Copilot assisted in the code, and what was learned through this project.
